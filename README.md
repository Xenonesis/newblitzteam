## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm 9+ or pnpm 8+

### Installation

```bash
# Clone the repository
git clone https://github.com/Xenonesis/newblitzteam.git

# Navigate to project
cd newblitzteam

# Install dependencies (choose one)
npm install
# or
pnpm install
# or
yarn install

# Start development server
npm run dev
```
